Guia de inicio Git
  1. Crear una carpeta en la pc, puede ser en el escritorio, por ejemplo MyProyecto_01
  2. Abrir con el editor de código (o un IDE tipo VSCode) el proyecto, agregar un archivo HTML, por ejemplo Prueba1.html, guardar.
  3. Para inicializar el proyecto, con Git Bash. Nos posicionamos sobre la carpeta. Pulsar Botón derecho del mouse : elegir opción Git Bash here.
  4. Si ejecutamos el comando ls nos muestra el contenido de la carpeta
  5. Para saber en que directorio nos encontramos : escribir pwd y presionar enter
  6. Para iniciar un repositorio local, ejecutar: git init (crea un nuevo respositorio git sobre el proyecto
  7. Vincular el repositorio a una cuenta con el nombre del desarrollador, asi cuando se realizan commit queda registrado 
                 git commit –global user.name “xxxx@xxxxxx.com.ar”
             git commit –global user.name “NOMBRE”
  8. Para ver en que estado están los proyectos del directorio, ejecutar: git status
  9. Para agregar el archivo prueba1.html al repositorio local ejecutar git add Prueba1.html
  10. Para verificar el estado del repositorio al ejecutar git status, crear un primer punto de partida ejecutando git commit -m “Mi primer commit”
  11. Para ver los cambios del commit, ejecutar: git log

Creación de Repositorio Remoto en GitHub
En la página GitHub - https://github.com
  1. En Mis repositorios -> New (nuevo) -> Nombre (aquí por ejemplo poner RepoPrueba). Tambien se puede agregar una descripción y se puede marcar dicho repositorio como público
  2. Ahora hay que traer el proyecto que creamos anteriormente y asi compartirlo.

    Ejecutar: git remote add origin https://github.com/nombreUsuarioGitHub/RepoPrueba.git (esta línea la obtenes de los comandos que nos ofrece la app)

    Pegamos dicho código en la consola y queda relacionado el proyecto a un repositorio remoto. Esto se ejecuta una sola vez!

  3. Luego, para subir los cambios hechos en el repositorio local, con la siguiente instrucción: git push -u origin master (o main si está definida con este último nombre). Si se ingresa a Github se puede ver el repositorio remoto con el contenido agregado.

PARA CLONAR UN REPOSITORIO
  1. 1) Necesitas que no exista el proyecto en la compu
  2. 2) Vas a github a la rama develop ir a code -> y te copias la dirección
  3. 3) Abrir git bash, posicionarse sobre el directorio donde vas a trabajar.
  4. 4) Ejecutar: git clone https://xxxxxxxxxx (esta dirección es la que copias desde github)
  5. 5) En git bash abrir el directorio con cd NOMBREDELDIRECTORIO
  6. 6) Para saber que ramas tengo disponibles git branch
  7. 7) Si necesitas crear una rama git branch develop
  8. 8) Para seleccionar la rama creada y poder trabajar en ella git checkout develop

Ahora si realizas cualquier modificación en VSCode sobre cualquier archivo del proyecto y ejecutas

git status (vas a poder ver que se realizaron cambios)

Entonces lo agregamos al repositorio local

git add prueba1.html (por ejemplo)

Si queremos agregar varios archivos

git add .

Para realizar commit

git commit -m “Mi Segundo commit”

Para fusionar la rama develop y la rama master, Pensando que estamos trabajando en develop:

        git checkout master
        git merge develop
        para subir todo a github:
        git checkout master
        git Branch
        git push -u origin master

Actualizar el repositorio GITHUB y ver los cambio
