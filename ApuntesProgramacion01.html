Arrays en Angular

En Angular se pueden crear arrays de varias formas. En esta sección se presentarán dos formas: declaración de un array literalmente y usando el método push para agregar elementos.

Declaración de un array literalmente

myArray : number[] = [1, 2, 3, 4, 5];

myArrayTexto : string[] = ["texto_01", "texto_02", "texto_03"];

Uso del método push() para agregar elementos a un array vacío

Declaración de un array vacío

myArray : number[] = [];

Agregando elementos con push()

this.myArray.push(1);

this.myArray.push(2);

this.myArray.push(3);

this.myArray.push(4);
