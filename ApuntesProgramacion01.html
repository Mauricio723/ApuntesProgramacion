Apuntes GIT / GITHUB
Guia de inicio Git
  1. 1) Crear una carpeta en la pc, puede ser en el escritorio, por ejemplo MyProyecto_01
  2. 2) Abrir con el editor de código (o un IDE tipo VSCode) el proyecto, agregar un archivo HTML, por ejemplo Prueba1.html, guardar.
  3. 3) Para inicializar el proyecto, con Git Bash. Nos posicionamos sobre la carpeta. Pulsar Botón derecho del mouse : elegir opción Git Bash here.
  4. 4) Si ejecutamos el comando ls nos muestra el contenido de la carpeta
  5. 5) Para saber en que directorio nos encontramos : escribir pwd y presionar enter
  6. 6) git init (crea un nuevo respositorio git sobre el proyecto
  7. 7) Vinculamos el repositorio a una cuenta con el nombre del desarrollador, asi cuando realizamos commit queda registrado git commit –global user.name “xxxx@xxxxxx.com.ar” git commit –global user.name “NOMBRE”
  8. 8) Ahora vemos en que estado están los proyectos del directorio con git status
  9. 9) Vamos a agregar el archivo prueba1.html a nuestro repositorio local con git add Prueba1.html
  10. 10) Si queres verificar el estado volves a ejecutar git status Vamos a crear un primer punto de partida git commit -m “Mi primer commit”
  11. 11) Con git log se pueden ver los cambios del commit
Creación de Repositorio Remoto en GitHub

AHORA NOS VAMOS A UN REPOSITORIO REMOTO -> GITHUB

En la página GitHub - https://github.com
  1. 1) En Mis repositorios -> New (nuevo) -> Nombre (aquí por ejemplo poner PRUEBA). Tambien se puede agregar una descripción y se puede marcar dicho repositorio como PUBLICO
  2. 2) Ahora tenemos que traer el proyecto que creamos anteriormente y asi compartirlo.

    Ejecutamos: git remote add origin https://github.com/userxxxxx/Prueba.git (esta línea la obtenes de los comandos que nos ofrece la app)

    Pegamos dicho código en la consola y queda relacionado el proyecto a un repositorio remoto. ESTO LO EJECUTAMOS UNA SOLA VEZ!

  3. 3) Luego vamos a copiar los cambios con la opción git push -u origin master (o main si la definiste con este último nombre). Si vas a Github podras ver el repositorio remoto con contenido.

PARA CLONAR UN REPOSITORIO
  1. 1) Necesitas que no exista el proyecto en la compu
  2. 2) Vas a github a la rama develop ir a code -> y te copias la dirección
  3. 3) Abrir git bash, posicionarse sobre el directorio donde vas a trabajar.
  4. 4) Ejecutar: git clone https://xxxxxxxxxx (esta dirección es la que copias desde github)
  5. 5) En git bash abrir el directorio con cd NOMBREDELDIRECTORIO
  6. 6) Para saber que ramas tengo disponibles git branch
  7. 7) Si necesitas crear una rama git branch develop
  8. 8) Para seleccionar la rama creada y poder trabajar en ella git checkout develop

Ahora si realizas cualquier modificación en VSCode sobre cualquier archivo del proyecto y ejecutas

git status (vas a poder ver que se realizaron cambios)

Entonces lo agregamos al repositorio local

git add prueba1.html (por ejemplo)

Si queremos agregar varios archivos

git add .

Para realizar commit

git commit -m “Mi Segundo commit”

Para fusionar la rama develop y la rama master, Pensando que estamos trabajando en develop:

        git checkout master
        git merge develop
        para subir todo a github:
        git checkout master
        git Branch
        git push -u origin master

Actualizar el repositorio GITHUB y ver los cambio
